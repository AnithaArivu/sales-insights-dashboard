# Project Title: Sales Insights Dashboard

## 📌 Overview
This project analyzes sales data using SQL, Tableau, and AI forecasting. 
It demonstrates skills in data cleaning, visualization, and predictive analytics.

## 🛠 Tools & Technologies
- SQL (data cleaning and queries)
- Tableau (interactive dashboards)
- Python (AI forecasting)
- OpenAI/Hugging Face (machine learning models)

## 📂 Dataset
- Source: [Kaggle Superstore Sales Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset)  
- Contains information about orders, sales, profit, and regions.

## 🔑 Steps Performed
1. **Data Cleaning**: Removed duplicates, handled null values, normalized tables using SQL.  
2. **Visualization**: Built Tableau dashboards showing KPIs (sales, profit, top products, regional trends).  
3. **AI Forecasting**: Used Python to forecast next month’s sales using regression/AI models.
